Welcome To My Demo Website

My name is Cavin Ngugi and this is my demo website. I have created this site to showcase the skills I have picked up from my time studying in Moringa School. I have also included a CSS framework known as bootstrap to make it look more stylish with less code.

In my time in Moringa school we have been taught various technologies and how to use them. A good example would be: -

  1. Version Control Systems i.e. Git
  2. HTML
  3. CSS
  4. JavaScript

This information was taught in only a span of 2 weeks! Welp they weren't kidding when they said it was a coding bootcamp. From these 2 weeks we have all spent some sleepless nights trying to finish assignments on time while also debugging.

That's a little joke right there :)
